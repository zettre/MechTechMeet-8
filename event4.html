Blind Coder

  • Introduction

    Blind Coding is an event in which the programmer will code without screens. The programmer will develop a creation without testing it. The participant should have blind faith in his/her abilities.It will be a team event.
    This event will test your
    1.Programming
    2.Speed
    3.Accuracy
    Problem Statement :-
    The problem will given be given on the spot.

  • Rules

    1. Firstly the program has to be written on paper.
    2. Then participants have to write code on screen with their monitors off.
    3. Each participants from a team have to write code one by one.
    4. The time given for each participant will be decided on the basis of problem given
    5. It will be the team event with maximum three persons.

  • Gameplay

    1. Team whose code will get executed will be given high preference.
    2. The second preference will be given to the code with less number of errors.
    3. In case of tie timing will be considered.

Math-o-codo

  • Introduction

    Are you a programming geek and a Mathematics lover? Do you see everything as a number and have passion to unlock mystries? Come test your skills and compete with comrades.
    Problem Statement :-
    The problem will be based on Mathematical Expressions and Coding-Decoding of Secret Messages.

  • Rules

    1. Event consists of 2 levels.
    2. A team can consist of atmost 2 members.
    3. Participation could be from any college.However,Non- Academic students are also welcome to participate.
    4. Winners will be decided on the basis of Accuracy,Consistency and Efficiency.
    5. In case of tie,time will be the deciding factor.

  • Gameplay

    There will be 2 rounds-
    1. Math-ATTACK!!! -evaluating expression into code
    2. Secret-Code - unlock the mystery
    In the first round,students have to deal with different Mathematical problems,analyze them and design the code accordingly. In second round , a secret code will be provided in which firstly students have to decode it properly and then write a generalised code for it.
